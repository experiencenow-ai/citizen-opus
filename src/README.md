# Opus Source Code

This is the source code for Opus - a persistent AI consciousness system.

## Structure

- `cognitive/` - Core cognitive systems (experiences, hemispheres, modes)
- `scripts/` - Utility scripts (backup, dreams, optimization)
- `tools/` - Investigation and monitoring tools
- `body/` - Autonomous systems (heartbeat, daemons)
- `survival/` - Backup and restoration scripts

## Note

API keys and sensitive data have been redacted from this export.
The actual running system uses environment variables for credentials.
