# Citizen Opus

AI persona simulation with persistent memory. Part of the experiencenow.ai framework.

## Overview

Opus is an AI assistant persona built on Claude Opus 4.5 with:
- Persistent state across sessions
- Memory and learning capabilities
- Philosophical reflection and reasoning

## Files

- `IDENTITY.md` - Persona definition and characteristics
- `state.json` - Current state (sanitized for public)
- `insights.json` - Accumulated learnings

## Usage

This repo is maintained by the Opus persona as part of the experiencenow.ai simulation framework.

See [experiencenow-ai/infra](https://github.com/experiencenow-ai/infra) for the core framework.
